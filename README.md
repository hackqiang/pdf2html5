# pdf2html5

##### running environment
		Ubuntu 14.04.3 LTS
##### required packgaes
		sudo apt-get install imagemagick inkscape pdf2svg

#### usage
		./pdf2html.sh test.pdf [pages]
###### or
		./batch.sh folder_contains_pdf [pages]

##### log are located in `log`
##### generated SVG are located in `tmp`
##### generated html are located in `output`
